# AI_Face_Detect
AI Face Authenticity Anti-Forgery Detection System

## Project Introduction
High-precision deep learning AI face forgery detection project.
The detection methods keep pace with and lead the latest AI development trends, with comprehensive multi-dimensional identification algorithms.
The overall face authenticity detection accuracy exceeds 99%.

## Main Functions
- Intelligent identification of AI-generated fake face images
- Multi-modal tampering face detection
- Friendly graphical interactive GUI interface
- Lightweight and fast reasoning detection speed

## File Structure
- main.py: Program main entry
- gui.py: Visual interface operation logic
- detector.py: Core face detection algorithm
- config.py: Project parameter configuration
- README.md: Project detailed description
